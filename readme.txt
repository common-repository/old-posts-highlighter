== Warnings ==

**Do not use Old Posts Highlighter if your <strong>permalink structure contains dates</strong>**

== Description ==

This module will <strong>randomly choose an old post</strong> in your Wordpress database and <strong>reset his publication date</strong>.

It will highlight older posts by moving them back to <strong>front page and in the RSS feed</strong>.

Old Posts Highlighter will improve your <strong>blog visibility, trafic and SEO</strong> !

You have the possibility to filter some categories, and to override this filter for specific posts.

Moreover, it will help you to share your post on social networks again, using specific tools detailled in the module options.

<a href="http://www.mkh.fr/old-posts-highlighter/">Old Posts Highlighter</a> is developped by <a href="http://www.mkh.fr">MKH</a>.

  
== Installation ==

This section describes how to install the plugin and get it working.

1. Upload Old posts Highlighter to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Optionally adjust the options

== Frequently Asked Questions ==

= Does this plugin generate duplicate content ?  =

No. The original post has the publication timestamp changed. There remains only only copy of the post. So no duplicate content ;)
